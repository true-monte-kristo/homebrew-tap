# True-monte-kristo Tap

## How do I install these formulae?

`brew install true-monte-kristo/tap/<formula>`

Or `brew tap true-monte-kristo/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "true-monte-kristo/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
